🛠️ פתרונות ויישומים

🏗️ בניית קהילת משמעות ▼
עקרונות יסוד
הקהילה אחראית להעניק זכות למשמעות
מימוש המשמעות הוא עניינו של הפרט
קהילה נבחנת ב"דקדוק היום-יום" - שגרות וטקסים
משאבים קהילתיים: תמיכה, שפה, מעטפת קיומית
הכיתה כמרקם חברתי - דוגמה מעשית:
📋 אמנה כיתתית
💰 "חשבון הפקדות" - מערכת הכרה
🎯 מודל יחסים-משימה
👥 ועדות כיתתיות
🎤 "הבמה שלי" - דיבור בציבור
🤝 עיבוד אירועים משותף
🏆 "קערת מעשים טובים"
🌍 יוזמות קהילתיות
👑 מנהיגות מחוללת משמעות ▼
תפקיד המנהיג

אחראי למשמעות של אחרים
בונה "שמיים" - מרחב לפעולה
מעצב את הקהילה כמחוללת משמעות
מנהיגים קהילתיים

מנהלים בעבודה
